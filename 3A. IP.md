## 🔹 1. **IP:** 192.168.10.77/**Маска:** 255.255.255.192

## Находим адрес сети (AND)

```text
IP:    11000000.10101000.00001010.01001101
Маска: 11111111.11111111.11111111.11000000
------------------------------------------------
Сеть:  11000000.10101000.00001010.01000000
```
## Переводим сеть в 10СС
```text
11000000₂ =
1·2^7 + 1·2^6 + 0·2^5 + 0·2^4 + 0·2^3 + 0·2^2 + 0·2^1 + 0·2^0
= 1·128 + 1·64 + 0·32 + 0·16 + 0·8 + 0·4 + 0·2 + 0·1
= 128 + 64
= 192

10101000₂ =
1·2^7 + 0·2^6 + 1·2^5 + 0·2^4 + 1·2^3 + 0·2^2 + 0·2^1 + 0·2^0
= 1·128 + 0·64 + 1·32 + 0·16 + 1·8 + 0·4 + 0·2 + 0·1
= 128 + 32 + 8
= 168

00001010₂ =
0·2^7 + 0·2^6 + 0·2^5 + 0·2^4 + 1·2^3 + 0·2^2 + 1·2^1 + 0·2^0
= 0·128 + 0·64 + 0·32 + 0·16 + 1·8 + 0·4 + 1·2 + 0·1
= 8 + 2
= 10

01000000₂ =
0·2^7 + 1·2^6 + 0·2^5 + 0·2^4 + 0·2^3 + 0·2^2 + 0·2^1 + 0·2^0
= 0·128 + 1·64 + 0·32 + 0·16 + 0·8 + 0·4 + 0·2 + 0·1
= 64
```
Адрес сети: 
```text
192.168.10.64
```

## broadcast
```text
192.168.10.64

Берём последний октет адреса сети
64 = 01000000
Добавляем хосты
01111111 = 0·2^7 + 1·2^6 + 1·2^5 + 1·2^4 + 1·2^3 + 1·2^2 + 1·2^1 + 1·2^0 = 127
192.168.10.127
```
